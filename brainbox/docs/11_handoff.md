## 1. Developer Guide
- **Setup**: `npm install` -> `npm run dev`.
- **Structure**: `src/components`, `src/screens`, `src/services`.
- **Conventions**: Functional components, Tailwind only, Framer Motion for animations.
- **API**: Всички заявки през `src/services`. Ключове в `localStorage`.

---

## 2. Designer Guide
- **Philosophy**: Glassmorphism, Dark Mode First, Fluidity.
- **Foundations**: OLED Black background, Inter typography, 4px spacing unit.
- **Effects**: NeuralField (Background), AmbientLight (Focus).

---

## 3. Stakeholder Guide
- **Vision**: "AI Second Brain" за организиране на дигитална информация.
- **Key Features**: Centralized Library, AI Nexus, Workspace, Chrome Extension.
- **Roadmap**: Mobile integration (Q2), Real-time collab (Q3), Local AI (Q4).
