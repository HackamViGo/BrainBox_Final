# UI Glossary

| Термин | Описание |
|--------|----------|
| **Glassmorphism** | Дизайн стил, използващ прозрачност и blur за симулиране на стъкло. |
| **NeuralField** | Интерактивен фон с частици, символизиращ невронна мрежа. |
| **AmbientLight** | Меко фоново сияние, което се променя според избрания модел. |
| **AI Nexus** | Централен екран за избор на активен AI модел. |
| **Refine** | Инструмент за оптимизиране на промпти чрез AI. |
| **Workspace** | Визуална среда за организиране на информация чрез нодове. |
| **Gateway** | Карта за бърз достъп до специфична функция в Prompts Library. |
| **Crystal** | Бутон за специфичен режим на оптимизация в Refine изгледа. |
| **Rail** | Тесен страничен панел (под-меню) в Sidebar. |
